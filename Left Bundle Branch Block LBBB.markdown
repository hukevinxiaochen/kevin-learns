# Left Bundle Branch Block LBBB
## Criteria
* Broad monophonic R vs M-shaped vs Notched vs RS Complex in Lateral Leads (I, aVL, V5, V6)
* R-wave Peak Time > 60 ms in Left Precordial Leads (V5, V6)
* Q in absentia in Lateral Leads
* Deep S in V1
